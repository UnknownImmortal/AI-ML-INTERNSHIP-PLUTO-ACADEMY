# AI/ML & Data Analytics Internship Projects

## Overview

This repository contains projects completed as part of an AI/ML and Data Analytics Internship program. The projects focus on real-world data analysis, machine learning, exploratory data analysis (EDA), visualization, and business insights generation using Python and modern data science tools.

## Objectives

* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Generate actionable business insights
* Build and evaluate machine learning models
* Develop practical problem-solving skills using data

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab
* Git & GitHub

## Projects

### Project 01: Zomato Restaurant Data Analysis

Analyzed restaurant data from Zomato to identify trends related to restaurant distribution, customer ratings, cuisines, online delivery services, pricing, and customer preferences.

**Skills Applied:**

* Data Cleaning
* Exploratory Data Analysis
* Data Visualization
* Business Insights Generation

### Project 02: Heart Disease Prediction

Built and compared multiple machine learning models to predict heart disease using patient health data.

**Skills Applied:**

* Classification Modeling
* Feature Analysis
* Model Evaluation
* Machine Learning Workflow
