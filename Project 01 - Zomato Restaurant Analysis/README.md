# Project 01: Zomato Restaurant Data Analysis

## Project Overview

This project focuses on analyzing restaurant data from the Zomato platform to understand customer behavior, restaurant trends, pricing strategies, cuisine popularity, and service offerings.

The analysis was performed using Python and various data analysis libraries to generate meaningful business insights.

## Dataset Information

The dataset contains restaurant information including:

* Restaurant Name
* Country
* City
* Cuisine Type
* Average Cost for Two
* Online Delivery Availability
* Table Booking Availability
* Aggregate Ratings
* Customer Votes

## Project Workflow

### 1. Data Loading

* Imported restaurant dataset
* Imported country code dataset
* Merged both datasets

### 2. Data Cleaning

* Checked missing values
* Handled missing cuisine records
* Verified duplicate entries

### 3. Exploratory Data Analysis

* Country-wise restaurant distribution
* City-wise restaurant distribution
* Online delivery analysis
* Table booking analysis
* Price range analysis
* Customer rating analysis
* Cuisine popularity analysis

### 4. Data Visualization

Created multiple visualizations using:

* Bar Charts
* Pie Charts
* Histograms
* Count Plots

## Key Insights

* India has the highest number of restaurants listed on Zomato.
* New Delhi has the largest restaurant concentration.
* Most restaurants do not provide online delivery.
* Table booking facilities are limited.
* Budget-friendly restaurants dominate the platform.
* Customer ratings are generally moderate to good.
